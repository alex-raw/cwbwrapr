## cwbwrapr

Some convenience wrappers to integrate CWB and CQP calls into R.
No integrated installation.

See: http://cwb.sourceforge.net/

## Install

```
install.packages("remotes")
remotes::install_github("alex-raw/cwbwrapr")
```
